软管水压试验（ RGJ-1 ）器操作规程

1 人员素质要求

1.1 了解本设备结构性能，能处理工作过程中的简单故障。

1.2 掌握设备管理常识 , 熟知“三好”、“四会”、润滑管理“五定”内容。

1.3 必须经培训合格后，持证操作本设备。

2 开工前准备工作

2.1 必须按规定穿戴好防护用品。

2.2 检查各风阀、风管路无漏泄，风压正常，按规定给油润滑。

2.3 同步升降机构加注润滑油，保证润滑良好。打开电源总开关，确认按钮站上“总停”按钮处于放松位置，“手动 自动”转换按钮处于“手动”位置。

2.4 检查风源压力大于 600 kPa 后，打开水源总阀门。确保试验台水箱里水清洁透明，如需换水即打开试验台水箱放水阀门，待排净污水后关闭放水阀门，打开水箱注水阀门，将水箱注水至箱体 2/3 处。

2.5 将制动软管安装支架升起，确认制动软管安装支架上的老虎头接口胶垫完好无损坏。如有漏气损坏现象，应及时修理或更换。

2.6 检查压力表不超期，作用良好。

2.7 检查各部动作作用良好，操作灵活， 填写台帐记录。

•  工作中应遵守的规定

3.1 工作中操作者不得随意脱离工作岗位。

3.2 使用方法：

3.2.1 各按钮作用如下：

“ 总停 ”：按下此按钮可切断整个控制系统的电源，如操作失误，试验台出现意外动作时可按下此按钮。

“ 手动 自动 ” ( 转换开关 ) ：转换手动与自动工作状态，处于“手动”位置时试验台的各种试验动作可由按钮站上的按钮来控制；处于“自动”位置时试验台试验过程由工控机自动控制。

“ 电源指示 ” ( 指示灯 ) ：当试验台具有正常的 +24V 控制电压时，此指示灯亮。

“ 开始 停止 ” ( 带指示灯按钮 ) ：试验台系统处于自动运行状态时按下此按钮试验台开始自动试验。

“ 气压 卸荷 ” ( 转换开关 ) ：当试验台处于“手动”试验状态时，按钮旋转至“气压”可对制动软管进行手动加气压试验，工作压力为 600-700kPa ，保压时间由人为自由控制。旋转至“卸荷”位可在试验完毕后对系统进行卸荷。

“ 水压 供水 ” ( 转换开关 ) ：当试验台处于“手动”试验时，按钮旋转至“水压”可对制动软管进行进行手动加水压试验，工作压力为 1.0-1.1MPa, 保压时间由人为自由控制 , 旋转至“卸荷”位可在试验完毕后对系统进行手动卸荷。

“ 上升 下降 ” ( 转换开关 ) ：当试验台处于“手动”试验状态时，按钮旋转至“上升”可对制动软管安装支架进行上升，观察制动软管状态，此时试验台的“左旋 右旋”均为失效状态。按钮旋转至“下降”可对制动软管安装支架进行 下降操作，同时“左旋 右旋”操作转为可用状态。

“ 左旋 右旋 ” ( 转换开关 ) ：当试验台处于“手动”试验状态时，按钮旋转至“右旋”可对制动软管支架进行右旋，观察制动软管状态，此时试验台的“上升 下降”操作为失效状态。按钮旋转至“左旋”可对制动软管安装支架进行左旋操作，同时“上升 下降”操作转为可用状态。

3.2.2 手动试验：

3.2.2.1 使“手动 自动”转换按钮处于“手动”位置。

3.2.2.2 将制动软管安装支架升起，装好制动软管。

3.2.2.3 用“上升 下降”按钮控制支架的上升和下降，用“左旋 右旋”按钮控 制支架旋转。旋转“气压”按钮，开始加气压至 600kPa 以上，目测压力表压力，当达到规定压力后松开“气压”按钮至中位，旋转按钮至“卸荷”位，压力表显示试验压力归零；旋转至“供水”按钮，待压力达到 250kPa 后，至水压位，当达到规定压力后松开“水压”按钮至中位，保压 1 分钟，旋转按钮至“卸荷”位，观察试验台各动作是否正常，确保试验台机能稳定后开始试验。

3.2.2.4 其它操作及 测试 过程同原手动试验台。

3.2.3 自动试验

3.2.3.1 使“手动 自动”转换按钮处于“自动”位置。

3.2.3.2 打开计算机的电源，进入 Windows 界面后，双击“试验系统”图标，即可进入试验主界面。

3.2.3.3 同原手动试验台，装好制动软管。

3.2.3.4 在开始试验前，录入延时设置，如“气压保压”时间，“水压保压”时间 等。如已进行过数据输入，且编辑框内保留的数据与本次将要录入的数据相同时，可不必再进行修改。

3.2.3.5 在程序中可对延时时间进行预先输入设置，如图所示

3.2.3.6 被测制动软管己正确安装后，可将按钮站上“开始 停止”按钮旋转至开始位，也可单击试验界面右下角的“开始”按钮开始自动试验。

3.2.3.7 在试验过程中系统随时检查试验台各个阶段的压力变化，如发现意外系统将暂停并在屏幕上弹出对话框，报告出现的情况并提示操作者应如何处理，操作者可根据实际情况进行处理或停止试验。

3.2.3.8 系 统将根据所测数据自动判 定，如不合格系统将报告故障情况，请操作者选择处理。

3.2.3.9 观察试验台已完成试验全过程，即可更换被测制动软管并重复步骤 4.6 开始下一组制动软管的试验。

3.2.3.10 在试验过程中可通过控制软件实现手动试验的各个操作。

3.2.3.11 如想结束试验，可旋转按钮至“停止”位或单击屏幕右上角的程序关闭按钮退出试验系统，关闭计算机与电源总开关。

3.3 长期不用时，应打开排水控制阀 , 排净增压缸内的水。

3.4 定期排除各风缸的污物，以防腐蚀风缸。

4 操作注意事项

4.1 操作者必须严格遵守有关安全规定及交接班制度。

4.2 在自动试验过程中，操作者要观察每个制动软管是否已经正确的升起或落下，若动作出错，即停止试验，待排除故障后才可继续试验。

4.3 开始手动或自动试验后，严禁身体进入试验台活动部位，以防发生意外。

试验过程中如发生意外情况，可压下“总停”按钮。

4.4 进行手动试验前微机要退出自动试验状态。

4.5 在试验过程中更换制动软管时，为防止意外发生，应将“手动 自动”按钮转至“手动”位置，并将系统卸荷，确保压力表显示工作压力为 0MPa 后方可更换新制动软管继续进行试验。

4.6 如想停止某组制动软管的自动试验，单击屏幕右下角的“停止”按钮。继续试验时，单击“开始” 按钮。

4.7 在更换制动软管、工装或进行其它操作时，严禁碰撞传感器，要保持试验台水箱内的水清洁透明，池内无杂物，避免阻塞放水阀门。

4.8 试验台为高压试验设备，日常操作时严禁违规操作。

5 完工后应做的工作　

5.1 　工作完毕关闭风源。

5.2 清洗试验槽，拧开旋塞，把污水排出。

5.3 擦拭清扫设备，做好日常保养，做到设备整齐、清洁、润滑、安全。

5.4 每星期或试验台停用两天以上开工前，用干净的棉纱擦净试验台的油污。

5.5 每星期在旋转机构的各连接处滴少量润滑油。

5.6 每运行两星期检查气源三联件，如过滤器内存水过多，松开放水旋钮放水。

5.7 每运行一个月，检查气控管阀是否运转正常，有无泄露情况。

5.8 每运行一个月，检查试验台机体与各电磁阀安装座，如松动重新拧紧。

5.9 每运行一个月，检查电源线与各控制线的接头处，如松动重新拧紧。

5.10 认真填写设备运转记录。

•  出现以下问题由使用人处理

6.1 分水器、油雾器中存有积水 , 应及时排除。

6 . 2 紧固松动的连接件。
