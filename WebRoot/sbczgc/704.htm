摇枕侧架抛丸除锈机 设备操作规程

•  人员素质要求

1.1. 操作者必须熟悉本规程，了解设备结构性能，经培训考试合格，取得设备操作证后，方能操作设备。

2 作业前准备工作

2.1 必须按规定穿戴好防护用品。

2.2 工作前应必须在机器运转前对有运动的零部件按规定进行润滑，确保各部润滑良好。

2.3 工作前应检查各部螺丝有无松动，耐磨件和护板的磨损情况，关闭各检修门，检查除尘管道有无漏气，除尘器中滤袋有无灰尘或破碎，清除分离器中过滤筛上的积物，关闭供丸闸阀。

2.4 检查抛丸器及抛丸器电机上的固定螺栓，以及其他各部件连接处的螺栓，有无松动，并及时进行紧固。

2.5 检查抛丸器内耐磨件和抛丸室内护板的磨损情况是否满足要求。

2.6 检查各检修门是否关闭，电控供丸闸阀是否关闭。

2.7 检查除尘管道有无漏气现象，除尘器中滤袋有无灰尘或破碎。

2.8 检查分离器中过滤筛上有无积物并及时清除。

2.9 检查各限位开关状态良好，控制台各信号灯工作正常。

2.10 检查提升机的松紧情况和抛丸器的振动情况。

2.11 检查弹丸的磨损情况和各部轴承、链条、销轴等活动部位的润滑情况。

2.12 及时填写设备台帐记录。

•  作业中应遵守以下规定

3.1 手动作业顺序：

3.1.1 合上总电源开关及各个断路器和空气开关，旋转钥匙开关 1SA 控制电源接通，按下“ PLC 电源通”按钮，接通 PLC 电源，将 PLC 内部开关从“ STOP ”扳到“ RUN ”的位置（如果无须改动程序，此开关应一直保持在“ RUN ”的位置）。

3.1.2 按“手动报警”按钮，提示工作开始。

3.1.3 转动 2SA “手动 / 自动”开关位置于手动位置。

3.1.4 按下“除尘风机开”按钮，打开主风机，电流指示。

3.1.5 按下“分离器开”按钮，打开分离器，指示灯亮。

3.1.6 按下“提升机开”按钮，打开提升机，指示灯亮。

3.1.7 按下“横向螺旋开”按钮，打开横向螺旋输送器，指示灯亮。

3.1.8 按下“纵向螺旋开”按钮，打开纵向螺旋输送器，指示灯亮。

3.1.9 按下“悬链驱动开”按钮，打开悬链驱动电机，指示灯亮。

3.1.10 旋转 “上料停止器开关”按钮，停止器开始动作，阻住小车，进行上料。

3.1.11 上完料在确定左边大门开着的情况下，旋转“上料停止器开关” 按钮，停止器气缸回位，小车载着料进入室体抛丸工位。

3.1.12 旋转速度开关，选择清理速度。

3.1.13 当工件进入室体后，依次按下 1 － 8 号抛丸器按钮，打开抛丸器。

3.1.14 分别按下相对应的供丸闸开按钮，清理开始，相对应的指示灯亮。

3.1.15 清理工位吊钩所挂工件清理完毕，按下“供丸闸关”按钮，指示灯灭。

3.1.16 按下“抛丸器关”按钮，关闭抛丸器。

3.1.17 根据需要动作抛丸室停止器。

3.1.18 旋转 “下料停止器开”按钮，停止器开始动作，小车停止，进行卸料。

3.1.19 卸料完毕，旋转 “下料停止器关”按钮，停止器回位，小车到上料工位。

3.2 自动作业顺序：

3.2.1 将手动－自动按钮打到自动位，按动自动开启按钮。自动工作时装料和卸料要手动进行。

3.2.2 选择清理速度。

3.2.3 风机自动启动。

3.2.4 弹丸循环系统（分离提升螺旋）依次间隔 5 秒钟开起来。

3.2.5 上完料，旋转 “上料停止器开关”按钮，停止器气缸回位，小车载着料进入室体抛丸工位。

3.2.6 小车感应抛丸室接近开关，低速推进系统动作，并根据选择的速度进行。

3.2.7 抛丸室体前的光电开关检测到进入室体内的两台小车载料情况，确定抛丸器开的情况，然后相对应的供丸闸自动开启，清理开始。

3.2.8 感应接近开关后，待工件出室体后，供丸闸自动停止，延时一段时间，抛丸器自动停止。

3.2.9 小车出室体到卸料工位，旋转 “下料停止器开关”按钮，下料停止器动作，小车停止，开始卸料。

3.2.10 卸完料后，旋转 “下料停止器开关”按钮，小车转到上料工位。重复 3.2.3 － 3.2.10 步骤。

3.3 如果已经上料完毕，而在室体内的工件还没有清理完毕，可以按下“进料停止器开”按钮，让工件在室体内等候清理。室体内的工件清理完毕，从室体中出来后，按下“进料停止器关”按钮，等候清理的工件室体。

3.4 出现故障时应按紧急停止按钮，以便停机迅速处理故障。

3.5 作业中，发现机器有较大振动，应及时停车检查抛丸器偏重情况，对称地更换叶片。

3.6 作业中，听到报警信号应立即停机检查，排除故障后再开机工作。

4 操作注意事项 :

4.1 操作者必须严格遵守有关安全、交接班制度；

4.2 作业前应按电笛报警以便离开危险区。

4.3 严紧室体地坑内潮湿，以防弹丸生锈。

4.4 机器周围散落地面上的弹丸应及时清扫，倒入室体内，以防滑倒伤人。

4.5 室壁内锰钢护板磨损后应及时更换，以防弹丸击穿室壁飞出室外伤人。

5 作业后应做的工作　

5.1 手动工作完毕应按以下顺序进行停机。

5.1.1 按下与工作的抛丸器相对应的“供丸闸关”按钮，停止供丸。

5.1.2 按下工作的“抛丸器关”按钮，抛丸器停止工作。

5.1.3 按下“纵向螺旋停”按钮，纵向螺旋关闭。

5.1.4 按下“横向螺旋停”按钮，横向螺旋关闭。

5.1.5 按下“提升停”按钮，提升关闭。

5.1.6 按下“分离停”按钮，分离器关闭。

5.1.7 按下“悬链驱动停”按钮，悬链驱动关闭。

5.1.8 按下“除尘风机停”按钮，主风机关闭。

5.1.9 按下“除尘旋喷反吹开”按钮，把除尘器布袋里的灰尘排到外面， 2 － 3 分钟后，按下“除尘旋喷反吹停”按钮，停止旋喷反吹电机。

5.2 自动工作停止后，按下“自动工作停止”按钮，整机将按照手动停机顺序自动延时停止。

5.3 整理作业中所用物品、工具。

5.4 清扫擦拭设备，做到洁净无污物，达到干净、整洁、润滑、安全。

5.5 做好设备工作运转记录。

6 出现以下问题由使用人处理

6.1 发现弹丸磨损不足，应及时补充。

6.2 紧固各部松动的连接件。

6.3 定期给各润滑部位加注润滑油，确保润滑良好。

6.4 气源体中如存有积水应及时排放。
