l 岗位名称

车辆配件焊修

2 目的和范围

2.1 适用于货车铸钢、锻钢配件磨耗堆焊及裂纹焊修。

2.2 对所用交流弧焊机进行操作保养。

3 任职要求

3.1 应达到中技专业技术水平，并通过国家劳动部职业技能鉴定，取得资格证书。

3.2 必须是在本岗位培训后，经教育科考试合格持有《上岗 证》。对需要持有设备操作证方能操作的设备，必须同时取得《设备操作证》，方可进行本工序工作。

4 工具、量具和工装设施

工作台、胎卡具、电焊设备、机床、焊修样板、预热设备、排烟和避光装置。

5 环境保护要求

5.1 本岗位必须做到环境整洁、物品定置良好。

5.2 焊修场地应设在 5oC 上静止空气中，电焊烟尘处理装置达到排放标准，避光装置作用良好。

5.3 将工作中产生的焊渣、废焊条、废配件等生产垃圾清理干净，送到规定的处理场所。

6 工序及技术要求

6.1 焊前检查

6.1.1 清除施焊表面（离焊缝边缘小于 20mm 宽度范围内）的油垢及氧化物。

6.1.2 鉴别材质，按材质选择适当焊条（鉴别配件材质：用外观、看标号、火花或理化化验均可）。

6.1.3 检测磨耗堆焊尺寸。

6.1.4 检测裂纹长度，如超过规定不得施焊（配件有裂纹时，应先焊裂纹，然后堆焊）。

6.2 焊前准备

6.2.1 根据焊件厚度、直径、坡口形式选用适当直径焊条（参见附录 1 ），按焊条直径调整电流（焊条应保持干燥）。

6.2.2 加金属填料时，材质应与被焊件材质相符，要适当增大电流保证熔深不小于 2mm （钩尾框、钩舌、钩尾销等配件不得加填充材料）。

6.2.3 焊修场地应在 5 ℃ 以上静止空气中。

6.2.4 锻钢件应在裂纹末端钻 8 ～ l 0mm 的止裂孔。

6.2.5 裂纹处开 60 ～ 70 °坡口（ V 型或 X 型），坡口深度和长度根据裂纹情况而定，以消除裂纹痕迹为准；用乙炔切坡口时，应彻底清除坡口溶渣和氧化物，露出基本金属。

6.2.6 圆钢制作的配件焊接时，两段间开 45 ～ 55 °或“ X ”型坡口（坡口形状如图 1-1 所示）。

图 1-1 园钢配件对接焊 X 型坡口

6.2.7 裂纹焊修后易产生毛细裂纹者，焊前用预备热炉或乙炔焰局部预热 250 ～ 300 ℃ 。

6.3 焊修

6.3.1 铸钢配件磨耗堆焊

6.3.1 .1 一般由左向右施焊，焊波宽应为焊条直径的 2 ～ 3 倍。

6.3.1 .2 焊第二条焊波前，应将第一条焊波的溶渣彻底清除掉，焊波相互重叠 30 ％以上。（待焊件暗红色完全消失后，方可消除溶渣）。

6.3.1 .3 电弧中断时，应在弧坑前引弧，不得在弯角处落弧。

6.3.1 .4 灭弧前，采取横向摆动焊条等方法填满弧坑。

6.3.1 .5 同一配件相应对称面焊补时，不得焊偏。

6.3.1 .6 在较大面积上施行多层焊时，为防止扭曲变形用逆向分段法施焊，应先后交叉 45 °左右，或 90 °分层焊。

6.3.2 铸、锻钢配件裂纹焊修

6.3.2 .1 一般由裂纹末端开始施焊，裂纹长度超过 150mm 时，采用逆向分段焊。

6.3.2 .2 施焊第一层焊波时，须采用适合坡口的焊条，充分焊透。

6.3.2 .3 不得在铸件的非焊修面引弧，以免电弧击伤铸件表面。

6.3.2 .4 由下向上逐层施焊时，要彻底清除前一层焊波溶渣，焊波接头要错开 20mm 以上。

6.3.2 .5 裂纹焊后，为防止焊缝裂纹，可覆盖干燥的石棉毡缓冷。

6.3.3 圆钢制的配件焊接

6.3.3 .1 沿坡口方向分层平焊，留 l ～ 3mm 加工余量。

6.3.3 .2 两段对接中心偏差不得超过 0.5mm 。

6.3.3 .3 由下向上逐层施焊时，要彻底清除前层焊波溶渣。

6.3.3 .4 焊后加热至 800 ℃ 以上，用胎具调修。

6.3.3 .5 调修后进行加工以恢复原型。

6.3.4 裂纹焊后按配件焊后正火工艺标准进行热处理（可立即覆盖石棉毡缓冷至室温）。

6.4 检查质量

6.4.1 堆焊后焊波凸凹不平高低差不得超过 2mm 。

6.4.2 堆焊后有扭曲变形者，应加热调修。

6.4.3 焊波与焊波、焊波与基本金属必须充分溶合。

6.4.4 焊波和焊缝不得有裂纹、夹渣、气孔、缺肉及未焊透等缺陷。

6.4.5 咬边深度不得超过 0.5mm 。

6.4.6 焊波、焊缝应有 1 ～ 3mm 的加工余量（以图纸基本尺寸确定加工余量）或增强焊波。

6.4.7 按规定需要在焊修时进行热处理的配件，可在热处理后进行质量检查。

6.4.8 根据配件的特定要求进行电磁探伤和拉力试验，不得有裂纹或变形。

6.4.9 杆件进行拉力试验，拉力为 11.7kN ／ cm 2 。

6.4.10 填写质量记录及 HMIS 录入

6.4.10 .1 作业完毕后认真填写相关质量记录。填写须齐全、准确、真实，字迹清晰。

6.4.10 .2 按 HMIS 岗位录入要求，及时登录相关工位模块，录入各项 HMIS 信息，检查无误后签名确认。

7 职业健康安全要求

7.1 班前班中严禁饮酒，班前要充分休息。

7.2 工作前必须按规定穿戴好防护用品，严禁穿拖鞋、高跟鞋。

7.3 电焊机必须绝缘良好，电焊变压器的一次绕组与二次绕组之间，引线与引线之间，绕组与引线外壳之间，其绝缘电阻均不得少于 1 兆欧。

7.4 电焊机变压器一次电源线，应使用绝缘良好的软线，一般不超过 2 ～ 3 米 ，因临时需要，可采取较长电源线，但应当高架离地面 2.5 米 以上使用瓷瓶挂线，禁止将电源线拖在地上。

7.5 电焊机一次或二次线 ( 把线 ) 应彻底绝缘，如短线接长时，接头最多不超过两个，但需按电工工艺接好。

7.6 电焊工作物和金属焊台同地相隔时需有保护性接地。

7.7 手钳线与焊条夹钳连接处，橡胶外皮应用一段深入到钳柄内部，导体不得外露，夹钳连接的把线，外皮及绝缘必须良好。

7.8 电焊机外壳必须有良好的接地保护或接零线，禁止放在高温或潮湿的地方。

7.9 施焊时应穿绝缘鞋 ( 靴 ) 领口用毛巾封闭，禁止穿潮湿的工作服进行焊接工作，施焊或更换焊条时，必须带手套。

7.10 工作地点周围不许有易燃品，并需与氧气保持 10 米 距离，由于场地限制，保证不了时，应使用铁罩遮挡。

7.11 焊修作业时，应及时利用自然通气或机械通风设备排换室内空气，将有毒气体排到室外，有通风除尘设施的作业点必须开启使用减少对身体的危害。

7.12 在高处施行焊接作业时，必须使用四脚梯子及紧固的支架，根据具体情况要戴好安全带，并通知下部附近的工作人员，防止火花烧伤。

7.13 电焊作业须由考试合格并持有操作证的熔接工进行，严禁乱拉乱用。

附件 2 机车车辆修理焊接技术条件

附 2.1 适用范围

适用于机车车辆及轨道起重的结构焊修。

本标准不适用于机车车辆用锅炉及压力容器的焊修。

附 2. 2 基本要求

附 2. 2.1 目前车辆修理中采用的焊接方法主要有焊条电弧焊、气体保护焊、埋弧焊 3 种。

附 2. 2.1 .1 焊条电弧焊具有方便、灵活、设备简单和适应性强等优点，在一般情况下可以获得较好的焊接质量。车辆检修中大多数使用焊条电弧焊方法。

附 2. 2.1 .2 气体保护焊主要有 CO2 气体保护焊、富氩气体（ Ar80%+CO220% ）保护焊，具有生产效率高、焊接质量好、焊接变形和应力小等优点。在车辆制造中已经普遍采用了气体保护焊。在车辆修理中，如磨耗件的堆焊和铸铁件的焊补也采用了气体保护焊。

附 2. 2.1 .3 埋弧焊主要用于中厚板长焊缝以及抗磨损材料的堆焊，具有生产效率高、成本低、焊接质量好、劳动条件好等优点。

附 2. 2.2 坡口型式

附 2. 2.2 .1 通常有 6 种，即 I 形坡口、 V 形坡口， X 形坡口， Y 形坡口，双 Y 形坡口、 U 形坡口带钝边。

附 2. 2.2 .2 在焊接生产中，为了使焊缝焊透，减小变形，提高生产效率，减少材料和电能的消耗，应尽量减小坡口的断面积。根据上述原则，不同厚度的焊件应开不同型式的坡口：

附 2. 2.2 .2.1 6 mm 以下的钢板对接双面焊时，一般可以不开坡口。

附 2. 2.2 .2.2 当钢板厚度为 6 ～ 12 mm 时，应采用 V 型坡口。

附 2. 2.2 .2.3 当钢板厚度为 12 ～ 20 mm 时，可采用 X 型坡口。

附 2. 2.2 .2.4 当钢板厚度大于 20 ～ 40 mm 时，就应考虑采用双 Y 形或 U 型坡口。

附 2. 2.2 .2.5 当板厚大于 40 mm 时，就应考虑采用双 U 型坡口。

附 2. 2.3 焊工必须经有关焊接理论的学习和实际操作培训，并按有关规定考试合格后，方可上岗操作。

附 2. 2.4 焊接设备及辅助器具，应按维护保养要求，保持良好状态，以保证使用安全可靠。

附 2. 2.4 .1 电弧焊机的技术要求应符合 GB 81.18 的规定。

附 2. 2.4 .2 埋弧焊机的技术要求应符合 GB l3164 的规定。

附 2. 2.5 焊接质量保证的一般要求应符合 GB/T 12467 的规定。

附 2. 2.6 焊接质量保证对企业的要求应符合 GB/T 12468 的规定。

附 2. 2.7 耐候钢焊接应符合 TB/T 2446 的规定。

附 2. 2.8 二氧化碳气体保护焊应符合 TB l582 的规定。

附 2. 2.9 低碳钢点焊应符合 TB/T 2454 的规定。

附 2. 2.10 焊接与切割的操作安全应符合 GB 9448 的规定。

附 2. 4 焊前准备

附 2. 4.1 零部件截换、挖补、补强时，其焊修处的单件及组装尺寸应符合有关检修规程的规定。

附 2. 4.2 采用自动与半自动气割方法加工坡口时，气割面质量应符合 ZB J59002.3 中Ⅱ级标准规定。

附 2. 4.3 采用等离子弧切割方法加工坡口时，其切割质量应符合 TB/T 2455 的有关规定。

附 2. 4.4 采用手工切割或碳弧气刨方法加工坡口时，坡口加工后，须除去坡口表面熔渣或气刨挂渣，并且不允许有深达 2mm 及以上的局部凹陷，否则应进行整修。

附 2. 4.5 容易淬硬的钢材，一般应采用机械切割方法加工坡口。

附 2. 4.6 施焊部位及两侧各 10 ～ 20mm 范围内应清除水分、铁锈、油污等杂物，但不影响焊接质量的预涂底漆除外。

附 2. 4.7 焊剂使用前，应按焊剂使用说明书的规定进行烘干。

附 2. 4.8 焊修裂纹时，应将裂纹彻底清理干净，并根据板厚情况，将裂纹部位开出坡口。必要时，裂纹端部应钻止裂孔。其具体工艺应符合有关检修规程的规定。

附 2. 4.9 须加补强板的部位，应将补强板范围内的灰尘及锈垢等彻底清除。磨平焊缝余高，并将平面调修平整，使补强板与焊件间的间隙不大于 1 ． 5mm ，补强板与焊件结合面非封闭时应涂防锈漆。

附 2. 5 焊接

附 2. 5.1 一般要求

附 2. 5.1 .1 为确保施工安全和焊接质量不受影响，焊接应在防风、防雨的设施内进行。

附 2. 5.1 .2 焊修补强板时，应符合下列要求：

附 2. 5.1 .2.1 补强板的焊脚尺寸应根据补强板的厚度及所受应力状态，由有关人员设计确定。

附 2. 5.1 .2.2 面积较大的补强板一般应加塞焊孔。其具体工艺应符合有关检修规程的规定。

附 2. 5.1 .3 要求焊前预热的焊件，在焊接过程中的焊缝层间温度应保持不低于预热温度。

附 2. 5.1 .4 严禁焊件在承载和受撞击的情况下进行焊修。

附 2. 5.1 .5 焊件如有热处理 ( 如回火等 ) 要求时，应在产品图样或技术条件中明确规定。

附 2. 5.2 采用手弧焊焊修的基本要求：

附 2. 5.2 .1 焊接电流的选取应符合各型焊条生产厂家的推荐值。

附 2. 5.2 .2 为减少焊件变形，缩小焊接应力，应选择合理的焊接顺序与方法。如可选用对称焊、分段退焊法、由中间向两端焊，以及跳焊等方法施焊。

附 2. 5.2 .3 板厚大于或等于 6mm 的对接焊缝采用双面焊要求焊透时，当正面焊缝焊完后，必须进行清根，方可进行背面焊缝的焊接。

附 2. 5.2 .4 多层焊时，每焊完一层后应彻底清除熔渣 ( 发现裂纹或其它不允许的缺陷应清除 ) 后，方可进行下一层的焊接，上下两层焊缝的接头应相互错开 20 ～ 30mm 。

附 2. 5.2 .5 对于相交的焊缝，焊缝的起孤点应避开焊缝相交处或转角处 50mm 以上。

附 2. 5.2 .6 焊补由焊件边缘向内发展的裂纹时，应由裂纹的末端起弧焊接。

附 2. 5.3 对埋弧焊的要求：

附 2. 5.3 .1 焊接电流、电弧电压和焊接速度等应调至规定值，并保持稳定。

附 2. 5.3 .2 焊剂撤布要均匀。焊剂厚度应不小于 30mm 。

附 2. 5.3 .3 单面焊时，接缝底部一般应采用焊剂垫、铜垫板或石棉带等作衬垫，或采用手弧焊封底。

附 2. 5.3 .4 在焊剂垫上施行单面焊时，必须严格保证装配质量。

附 2. 6 对磨耗件堆焊的要求

附 2. 6.1 在较大的焊件进行堆焊时，应采取适当的工艺措施（包括使用必要的夹具及支撑铁，采用反变形，对称焊法、跳焊法，以及采用较小的焊接规范等），以减小焊件的变形。

附 2. 6.2 堆焊层硬度较高时，应采取适当的工艺措施（如对焊件进行整体或局部预热，避免连续多层堆焊，采用隔离层等），以减小焊接应力，防止堆焊层裂纹及剥离。

附 2. 6.3 堆焊中、高碳钢或有淬火倾向的低合金钢焊件时，为防止和减少焊件热影响区裂纹的产生，焊前应根据母材及堆焊焊条的要求进行预热。如母材所要求的预热温度与堆焊焊条不同，预热温度应取其中要求较高者。焊接时应采用较小的焊接电流，焊后应注意缓冷。

附 2. 6.4 在厚度不均匀的焊件上进行堆焊时，其堆焊的起点应放在较薄的一端，以避免堆焊处产生裂纹。

附 2. 7 质量检验

附 2. 7.1 焊件在检查验收前必须清除焊缝的熔渣和飞溅。焊接质量的检验工作一般应在油漆前进行。

附 2. 7.2 用肉眼或五倍放大镜检查焊缝表面的质量。

附 2. 7.2 .1 用尺或样板等检查焊缝的形状、尺寸及焊接缺陷的尺寸。

附 2. 7.3 焊接缺陷分类及说明按 GB 6417 的规定。

附 2. 7.4 焊缝外形尺寸应符合 GB l0854 的规定。

附 2. 7.5 焊接结构未注尺寸公差按 TB/T 2250 的规定。

附 2. 7.6 焊缝内部缺陷的检查，应根据产品技术条件或产品图样的规定，一般可采用射线照相、超声波探伤方法检查。

附 2. 7.6 .1 射线照相的检查，按 TB/T 2245 或 GB 3323 等标准的规定进行。

附 2. 7.6 .2 超声波探伤的检查按 TB l558 或 GB ll345 等标准的规定进行。

附 2. 7.7 对要求致密性的焊缝，可选用盛水、淋水、水压及煤油渗透等试验方法检查。

附 2. 7.7 .1 盛水试验方法及其要求应在产品图样或其技术条件中规定。

附 2. 7.7 .2 淋水试验方法及其要求应符合 TB l802 的规定。

附 2. 7.7 .3 水压试验及煤油渗透检查方法见附录 A 。

附 2. 7.8 焊缝金属及焊接接头的力学性能的试验应符合 GB 2649 ～ 2656 的规定。

附 2. 7.9 重要零部件的焊接，应按工厂工艺文件的要求，在规定位置打上焊工的钢印或登入厂内记录簿。

附件 3 机车车辆耐候钢焊接技术条件

附 3.1 主题内容与适用范围

适用于机车车辆用耐候钢的焊接。

附 3.2 技术要求

附 3.2.1 基本要求

附 3.2.1 .1 机车车辆用耐候钢部件焊接接头的基本型式与尺寸应符合 GB 985 和 GB 986 的规定。

附 3.2.1 .2 产品图样的焊缝符号及焊接方法在图样上的表示代号应分别符合 GB 324 、 GB 5185 和 GBl2212 的规定。

附 3.2.1 .3 焊工必须按铁道部“焊工考试办法”的规定经考试合格后，方可上岗操作。

附 3.2.1 .4 焊接设备应保持良好的状态。

附 3.2.1 .5 焊接操作安全应符合 GB 9448 的有关规定。

附 3.2.1 .6 耐候钢焊接接头的耐蚀性应相当于耐候钢母材的耐蚀性能。

附 3.2.2 焊接材料

附 3.2.2 .1 机车车辆所用耐候钢材，其型式尺寸与技术条件等应符合 TB 1979 或 GB 4171 、 GB4172 的有关规定。

附 3.2.2 .2 凡属进口耐候钢材应附有质量合格证书，必要时应进行焊接工艺评定或焊接性试验，合格后方可投入使用。

附 3.2.2 .3 凡第一次使用的耐候钢材应进行焊接工艺评定，合格后方可投入使用。

附 3.2.2 .3.1 焊接工艺评定按铁道部颁布的《零部件焊接工艺评定规则》的规定进行。

附 3.2.2 .3.2 焊接性试验推荐选用 GB 4675.1 ～ GB 4675.3 ， GB 4675.5 等标准规定的试验方法。

附 3.2.2 .4 手弧焊所用焊条及 CO2 焊所用焊丝，其牌号选用、技术要求、试验方法及验收规则等均应符合 TB ／ T 2374 的有关规定。

附 3.2.2 .5 埋弧焊时，推荐选用 H08CrNiCu 焊丝及 HJ431 焊剂或与母材相适应的其它焊接材料进行焊接。

附 3.2.2 .6 焊接所用 CO 2 气体，其纯度应不低于 99.5 ％，含水量应不超过 0.05 ％。

附 3.2.2 .7 普通碳素钢与耐候钢焊接时，推荐选用耐候钢焊接所用的焊接材料，也允许按其中较低强度的母材选用焊接材料。

附 3.2.2 .8 耐候钢板厚大于 12mm 时，推荐选用低氢型耐候钢焊条进行焊接。

附 3.2.3 焊前准备

附 3.2 ， 3.1 耐候钢焊件待焊部位及焊缝两侧各 20mm 范围内应清除油污、铁锈、水分、尘土等。但不影响焊接质量的预涂底漆除外。

附 3.2.3 .2 耐候钢焊接零部件，焊前需经检查人员检查，确认坡口各部尺寸及表面质量合格后方可施焊。

附 3.2.3 .3 耐候钢焊条因受潮而影，向焊接质量时，使用前应按说明书的规定进行烘干。

附 3.2.3 .4 耐候钢焊丝表面必需光滑、平整、清洁，对焊丝表面的油污及锈蚀必须清除干净。

附 3.2.3 .5 埋弧焊焊剂在使用前应按 GB12470 的规定进行烘干。

附 3.2.3 .6 采用自动与半自动气割方法加工坡口时，气割面质量应符合 ZBJ59 002.3 《热切割气割质量和尺寸偏差》中的Ⅱ级标准规定。

附 3.2.3 .7 采用等离子弧切割坡口时，其切割平面度 u 及割纹深度 h 应不低于 ZBJ59002.4 中的Ⅱ级标准。

附 3.2.3 .8 采用手工切割加工坡口时，须除去坡口表面的氧化熔渣和飞溅物，并且不允许有深达 2mm 以上的局部凹陷，否则应进行铲修。

附 3.2.3 .9 对接接头的边缘高度差应不超过 TB1580 中的表 1 规定。
