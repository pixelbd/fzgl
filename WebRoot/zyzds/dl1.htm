作业前准备及安全注意事项：

1. 架车前须分解脱轨自动装置拉环。

2. 班前班中严禁饮酒，班前要充分休息。

3. 工作前必须按规定穿戴好防护用品，严禁穿拖鞋、高跟鞋。

4. 工作前必须检查所使用的工具良好。

5. 架落车作业，须由专人亲自指挥，架车后由工长分别检查各铁马、架车机的安全状态及架车机安全销状态。

6. 架落车工作须由三人以上进行、架车前要由专人对架车使用的架车机及垫木、电镐及电线等 技术 状态进行认真检查。

7. 架车前或落下一个转向架后，均要打好 防滑楔块 ，起落车前应认真实行呼唤应答。架车时除架车人员外，车上车下不得有人，严禁他人在所架的车上和车下作业。

8. 使用天车架落车时，不得两端同时架落，不准多人同时指挥天车作业。

9. 使用架车机进行架落车时，不得两端同时架落，车架好后必须在架车机上安装安全销。

10. 在架落车中，取出和调整心盘销时，要使用专用工具并须在转向架两旁承上用安全木防护后，方准进行。

 

作业程序

测量钩高 → 分解脱轨自动装置拉环 → 分解移动杠杆与上拉杆连接圆销 → 架车 → 推出转向架 → 推入转向架 → 落车 → 调整钩高 → 调整旁承游间 → 安装脱轨自动装置拉环

架车

1. 分解移动杠杆与上拉杆的连接圆销，放于指定位置。

2. 放置好防滑木掩、架车机防滑垫板及插销、木楔等架车设备或工具，一人开风阀或指挥天车，另两人架起车体后插好架车机横穿销或放置好铁马，推出转向架，将车体两端分别平稳架起。

3. 架车作业时，车上、车下不准有人，架车之另一端转向架轮对外侧须打木楔防滑。

4. 架车机须有防滑垫，达到四点稳固（即四个架车机均与侧梁接触）。

5. 推出转向架，将转向架吊运至指定检修地点。

 

落车

1. 将有验收合格标记的转向架按规定路线响铃吊运，看准转向架顺位标记，平稳降落在相应车端轨面上。

2. 落车时心盘及心盘磨耗盘内不得有铁垫、开口销等杂物。

3. 转向架推入车体下部，拔出风动架车机横穿销（或撤除铁马），对准中心销，平稳落下车体，上心盘须落槽。中心销须落入下心盘座孔内。如心盘销与上心盘孔错位时，须以专用工具拔正，严禁用手扶心盘销落车。

 

落车测量有关尺寸

1. 上、下心盘之间的螺栓与铆钉垂直相对距离不小于 5mm ；上心盘底座平面与下心盘立棱间的距离不小于 3mm 。

2. 铅锤测量同一端梁上平面与轨面水平线的垂直距离左、右相差不大于 20mm 。

3. 在平直线路上检查上旁承组成下平面与下旁承滚子的间隙为 5 ± 1mm ，不符时可用下列方法调整：

3.1 下心盘加装垫板后，须先调整下旁承垫板，数量为 1~3 块，总厚度为 2~25 L。

3.2 调整下旁承垫板仍不符合要求时，可再调整上旁承垫板，数量为 1~3 块，总厚度为 2~35 L。

3.3 整车落成需调减心盘垫板厚度时，先相应调减或撤除上旁承调整垫板后重新按上述两条要求调整。上、下旁承加装调整垫板和下心盘加装垫板后，上旁承磨耗板下表面至上心盘下表面垂直尺寸及下旁承磨耗板上表面至下心盘上表面垂直尺寸（不含心盘磨耗盘）发生相应改变。

4. 车钩中心线至轨面的垂直距离：为 880 ± 10mm 。同一车辆两车钩中心高度差不大于 10mm 。

5. 17 型车钩上翘量不大于 5mm ，下垂量不大于 8mm 。车钩上翘、下垂量可更换支撑弹簧或支撑座磨耗板调整，磨耗板厚度为 4 ～ 16mm 。
