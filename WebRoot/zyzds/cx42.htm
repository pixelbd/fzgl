1 岗位名称

心盘加修前检测

2 目的和范围

2.1 负责心盘加修前检查，确定心盘的施修范围。

2.2 本指导书适用于货车各型上、下心盘的修前检查、测量。

3 任职要求

3.1 应达到中技专业技术水平，并通过国家劳动部职业技能鉴定，取得资格证书。

3.2 必须是在本岗位培训后，经教育科考试合格持有《上岗证》。对需要持有设备操作证方能操作的设备，必须同时取得《设备操作证》，方可进行本工序工作。

4 工具、量具和工装设施

手灯；手锤；下心盘磨耗检查样板；下心盘原型检查样板；上心盘磨耗检查样板；上心盘原型检查样板； 1m 钢直尺；塞尺。

5 环境条件

本岗位必须做到环境整洁、物品定置良好。

6 作业程序及标准

6.1 工作前的准备

必须将工作服、手套等劳保用品穿戴整齐，检查各种样板、量具齐全，检定不过期。

6.2 工作中的控制程序

6.2.1 心盘平面磨耗超过 3mm ，直径磨耗超过 1mm 时要求堆焊加工恢复原型（直径磨耗测量部位：下心盘由平面向上 5mm 起计算，上心盘由平面向上 15mm 起计算）。转 8AG 、转 8G 下心盘直径原型为 φ 308 + 0.95 -0.30 mm 。

6.2.2 脐部磨耗剩余厚度不足 7mm 时或脐部有裂纹时要求铲坡口或切除脐部，镶焊套筒。

6.2.3 筋部裂纹时要求焊修。

6.2.4 变形时要求加热调修。

6.2.5 上心盘安装孔局部损伤、铆钉孔直径大于 2mm ，平面深度大于 3mm 时，要求堵孔重钻。

6.2.6 改进型上心盘无内筋者无须加筋（改进型上心盘磨耗面两侧削斜 4 °，削斜宽度 25mm ）。

6.2.6 经检查符合检修条件的上、下心盘，在需加修的故障部位用白粉笔涂打加修标记；没有加修价值须报废的配件涂打报废标记，并转送到报废配件区。

6.3 工作后的处置

确认各物品摆放符合定置管理要求，打扫场地卫生。

6.4 样板量具使用说明：

6.4.1 转 K2 、 K4 上、下心盘样板使用说明：

6.4.1 .1 下心盘原型检测样板

样板用于下心盘加修后检测心盘直径和高度。用 354T 和 356Z 检测心盘直径， 354T 端通过， 356Z 端止住者为合格；用 32T 和 34Z 检测心盘高度，高度介于 32 与 34 之间者为合格。

6.4.1 .2 下心盘磨耗检测样板

用 354 检测心盘直径磨耗，能放入者为不合格；用 34 检测心盘平面磨耗，能放入者为不合格；

6.4.1 .3 上心盘磨耗检查样板

6.4.1 .3.1 尺寸“ 349 ” 用于检查上心盘直径磨耗，将样板套入上心盘内，若样板的底面能接触到上心盘的底面，则表明上心盘直径已磨耗过限 ;

6.4.1 .3.2 尺寸“ 43 ” 用于检查上心盘平面磨耗，将样板的贴靠上心盘的立棱顶面，样板的斜面与心盘立棱侧面贴靠，此时，若样板的与上心盘底面之间有间隙，则表明上心盘的平面磨耗未到限，若能与上心盘底面贴靠，而样板与上心盘立棱接触或有间隙，则表明上心盘平面已磨耗过限。

6.4.1 .4 上心盘原型检查样板

6.4.1 .4.1 上心盘直径 : 上心盘加修后，用尺寸“ 349Z ”、“ 351T ”检测其直径。“ 349Z ”侧应不能插入到底 ( 止住 ) ；用“ 351T ”侧检查能插到底 ( 通规 ) 者为合格。

6.4.1 .4.2 上心盘平面 : 上心盘加修后，用“ 351T ”侧检查其高度应介于 45mm 刻线与 47mm 之间者为合格。

6.4.2 转 8A 型上、下心盘样板使用说明：

6.4.2 .1 下心盘原型检测样板

该样板用于下心盘加修后检测心盘直径和高度。用 304T 和 306Z 检测心盘直径， 304T 端通过， 306Z 端止住者为合格；用 24T 和 26Z 检测心盘高度，高度介于 24 与 26 之间者为合格。

6.4.2 .2 下心盘磨耗检测样板

用 299 检测心盘直径磨耗，能放入者为不合格；用 28 检测心盘平面磨耗，能放入者为不合格；

6.4.2 .4 上心盘原型检查样板

6.4.2 .4.1 上心盘直径 : 上心盘加修后，用尺寸“ 299Z ”、“ 301T ”检测其直径。“ 299Z ”侧应不能插入到底 ( 止住 ) ；用“ 301T ”侧检查能插到底 ( 通规 ) 者为合格。

6.4.2 .4.2 上心盘平面 : 上心盘加修后，用“ 301T ”侧检查其高度应介于 45mm 刻线与 47mm 之间者为合格。

6.4.3 转 8AG 、转 8G 型上、下心盘样板使用说明：

6.4.3 .1 下心盘原型检测样板

该样板用于下心盘加修后检测心盘直径和高度。用 307T 和 309Z 检测心盘直径， 307T 端通过， 309Z 端止住者为合格；用 28T 和 30T 检测心盘高度，高度介于 28 与 30 之间者为合格。

7 职业健康安全要求

7.1 班前班中严禁饮酒，班前要充分休息。

7.2 工作前必须按规定穿戴好防护用品，严禁穿拖鞋、高跟鞋。

7.3 工作前必须详细检查所使用的工具作用良好。

7.4 检测心盘时，要注意是否有毛刺，注意心盘的码放高度，防止刺伤、砸伤脚手。
