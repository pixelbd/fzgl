工作前准备：

1. 工作者必须佩戴好劳保用品。

2. 全面检查油漆与标记涂打设备性能状态良好，动作正确。对本岗位所用设备和模板、量具进行操作保养。

3. 了解当日和次日入线车的情况，并按车辆出车顺序，刷去旧标记。

4. 按计划对新换修的木质配件和金属配件及经除锈、加热处理后的部位按其原色进行涂漆。

5. 按出车计划和预检员的标记进行除锈。

6. 按计划涂打车辆标记，并进行核对，以免重复作业。

 

作业程序：

涂抹原车辆标记 → 除锈 → 油漆 → 涂打车辆标记

 

除锈作业

1. 车体及底架锈蚀部位、油漆脱落处应铲除锈皮补漆。

2. 新换件的外侧面除锈后涂漆。

车辆油漆作业

1. 涂油漆处所应呈干燥状态，金属表面锈垢或熔渣须清除。

2. 涂漆应在焊铆件或加热调修，热处理充分冷却后进行。

3. 油漆中不得掺汽油、煤油以及防油漆粉化。

4. 油漆冬季允许加不超过 5% 的催干剂。

5. 不同种类的油漆，禁止掺对使用。

6. 防锈底漆涂刷后待干燥到不沾手时，方可涂面漆。

涂打车辆标记

抄写现车标记后涂盖原标记。待底漆干燥至不沾手程度时，方可

喷涂标记。

1. 深黄色面漆上标记颜色为黑色，其余为白色。

2. 厂修： 8 年，段修： 2 年，不涂打“取消辅修”标记。

3. DL1 型运梁车组中原为“ NX17K ”标记改为“ DNX17K ”，同时修改车辆标签相应内容。

4. DNX17K 车体上涂打“运梁车组专用车”标记。

5. 其它标记须分别符合图样要求。
