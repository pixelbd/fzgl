l 岗位名称

从板机加工

2 目的和范围

2.1 负责从板机加工，并负责对从板机加工质量进行检查。

2.2 对所使用的刨床等进行操作保养。

2.3 本指导书适用于从板的机加工。

3 任职要求

3.1 应达到中技专业技术水平，并通过国家劳动部职业技能鉴定，取得资格证书。

3.2 必须是在本岗位培训后，经教育科考试合格持有《上岗证》。对需要持有设备操作证方能操作的设备，必须同时取得《设备操作证》，方可进行本工序工作。

4 工具、量具和工装设施

从板刨床、从板综合检查样板。

5 环境条件

5.1 本岗位必须做到环境整洁、物品定置良好。

5.2 及时将刨削的废铁屑收集到材料箱中定期交材料科处理。

6 作业程序及标准

6.1 工作前的准备

必须将工作服、手套等劳保用品穿戴整齐，检查起重机、从板刨床等状态良好，各种样板、量具齐全，检定不过期。

6.2 工作中的控制程序

6.2.1 从板刨修

6.2.1 .1 将焊修后的从板吊到从板刨床夹具上，卡紧待加工从板，根据从板焊修长（厚）度调整至最佳刨修位置，保证刨修后的从板达到限度要求。

6.2.1.2 刨削进刀量不宜超过 3mm 。

6.2.1 .3 从板刨修后须达到原型尺寸（长 319 +3 -2 ×宽 225 +2.5 -1.5 ×厚 57 +1.5 -1.0mm ）。

6.3 检查质量：

6.3.1 从板刨修后用“从板原型检测样板”测量从板各部尺寸，须达到：长 319 +3 -2 ×宽 225 +2.5 -1.5 ×厚 57 +1.5 -1.0mm 的原型尺寸。

6.3.2 加工面须平整，不得有黑皮、毛刺。

6.3.3 机加工后不允许有不大于 0.5mm 的咬边、凹陷存在。

6.4 工作后的处置

6.4.1 关闭电源，按照 地控式起重机和从板刨床操作规程，达到整齐、整洁、润滑、安全。

6.4.2 确认各物品摆放符合定置管理要求，打扫场地卫生，将刨削的废铁屑收集到材料箱中定期处理。

6.4 样板量具使用说明：

6.4.1 从板综合检查样板：

6.4.1 .1 检测厚度 : 厚度用尺寸“ 58.5T ”检查，应均能插入 ( 通过 ) ，用尺寸“ 56Z ”检查，任一处均不能插入 ( 止住） ;

6.4.1 .2 检测宽度 : 宽度用尺寸“ 227.5T ”检查任一处，均应能插入 ( 通过 ) ，用尺寸“ 223.5Z ”检查均不能插入 ( 止住 );

6.4.1 .3 检测长度 : 长度用尺寸“ 322T ”检查任一处均能插入 ( 通过 ) ，用尺寸“ 317Z ”检查，均不能插入 ( 止住 ) 。

6.5 填写质量记录及 HMIS 录入

6.5.1 作业完毕后认真填写相关质量记录。填写时需做到字迹清晰、干净整齐、不涂不改、不错不漏，提倡采用仿宋体填写。

6.5.2 按 HMIS 岗位录入要求，及时登录相关工位模块，录入各项 HMIS 信息，检查无误后签名确认。

7 职业健康安全要求

7.1 班前班中严禁饮酒，班前要充分休息。

7.2 工作前必须按规定穿戴好防护用品，严禁穿拖鞋、高跟鞋。

7.3 刨床最大行程界限内，不准堆放任何物品，并须设固定或移动防护挡杆。 7.4 装夹工件要用合适的压板，以防移动。

7.5 刨床正在刨削工作物时不得变更速度不得用手调正工作物或触摸工作物表面，检查工件时，不要把头部放在刨床行程内及用手扶刀头上，必要时须停车检查。

7.6 开车时不准用手触摸，或用尺量工作物；要戴护目镜作业。

7.7 刨刀伸出于刀台部分的长度，不得大于刀刃的三倍。

7.8 在对刀时要注意计算速度、检查吃刀量和进刀量，当刨大工作物时，要事先对好行程，用手搬动往复一次，而后开车。
